Custom Web AppBuilder Widget: MultiFilter

The MultiFilter addresses limitations with the native Group Filter Widget by allowing multiple group layer filters to be applied to the web map at one time and allowing the creation of complex queries for each layer within a group.

Multi-Criteria Analysis for Environmental Code Inspector Boundary Redistricting

Visualizing Bike Share Connectivity Using the ArcGIS API for JavaScript

Visualizing the pick-up and drop-off locations of Blue Bike bike share customers in Boston, Massachusetts for the month of July, 2018.

This web map was built using the ArcGIS API for JavaScript 4.9 and is used to demonstrate how an understanding of the API can be leveraged to implement web map functionality that transcends Web AppBuilder for ArcGIS.

Geoprocessing Service: Solid Waste Collection Date Lookup

This geoprocessing service finds the Solid Waste pick up date for a given household and was used as a proof of concept to transition departments away from static data, such as Excel spreedsheets, and adopt dynamic data practices.

Python Geoprocessing Tool: Asbuilt Record Locator

This Python Geoprocessing Tool uses the ArcPy Python library to automate a frequent GIS task - finding and delivering asbuilts for customer requests.

When benchmarked against a request that previously took a day's work, this script was able to provide a 30-minute turnaround.
