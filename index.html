ArcGIS API for JavaScript 4.9: Open Source Feature Table Widget

Anyone migrating from ArcGIS API for JavaScript 3 might have noticed that the Feature Table Widget - a module for creating attribute tables in web maps - was not included in the latest release. Eager to have the same simplicity in version 4, I decided to create my own Feature Table Widget.
The Feature Table Widget performs a client-side query on a user defined layer and passes the results to a dojo store. The resulting attribute table can be filtered using the custom filter-popup, which relies on the RQL query engine.
Checkout my GitHub to download or contribute to the widget.
